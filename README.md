# Mohammed Dilshad P Portfolio

Premium light-theme developer portfolio built with Next.js and populated from `Mohammed_Dilshad_P_.pdf`.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS with custom design tokens and responsive layout

## Included content

- Hero introduction
- About / technical philosophy
- Categorized skill system
- Detailed project showcases for Alizo and NaDodi
- Supporting Adam cross-platform project
- Experience, education, languages, and contact CTA
- Resume download link
- Secure response headers via `next.config.ts`

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm run start
```

## Deployment

This project is ready for Vercel or any Node-compatible host that supports Next.js.

### Vercel

1. Import the repository into Vercel.
2. Keep the default Next.js build settings.
3. Deploy.

### Manual hosting

1. Install dependencies with `npm install`.
2. Build with `npm run build`.
3. Start the production server with `npm run start`.

## Notes

- All displayed professional details were extracted from the resume PDF and cross-checked against the existing project materials in the local workspace.
- Images and the resume PDF are stored locally in `public/assets`.
